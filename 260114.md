## Python 자료형

| 자료형 | 예시 | 비고 |
| :--- | :--- | :--- |
| int | a = 12 | 숫자 |
| bool | a = True | 숫자 |
| str | a = '안녕' | 문자열 |
| list | a =  [1,2,3,4,5] | 리스트(배열) |
| tuple | a =  (1,2,3,4,5) | 고정형 배열(내용 변경 X) |
| set | a =  {5,4,3,1,2,3,4,5} | 중복값을 덮어쓰기한 리스트 |
| dict | a =  {'key':'value','age':'28'} | 객체(딕셔너리, js에서의 object) |

### str, list, tuple : 시퀀스 
- a [ i : j ]로 인덱싱 및 슬라이싱 가능

### set과 dict은 중괄호({ })를 사용. 
- set이 중복값을 덮어쓰기한 것처럼 dict도 key값이 중복되면 마지막 value를 덮어쓰기.
- 내용물의 순서를 저장하지 않기 때문에 a [i] 로 값을 찾을 수 없다.
- 두 개 이상의 객체를 & , | , - (빼기) 로 교집합, 합집합, 차집합을 만들 수 있다.
- set은 순서가 1,2,3 / a,b,c 순으로 정렬됨

```py

a = {5,6,4,1,2,3,4,5}
b = {4,7,0,3,1,9}

print(a&b) ==> {1,3,4}
print(a|b) ==> {0, 1, 2, 3, 4, 5, 6, 7, 9}

```

## for 반복문

### for i in a: a의 요소값 만큼 반복
```py
a = ['안녕','난','김지환','이야']
for i in a :
    print(i)

# '안녕'
# '난'
# '김지환'
# '이야'

```
- a의 length만큼 반복문을 실행하고 싶다면?

### len(a)을 사용하여 a의 길이 찾기

```py
a = ['안녕','난','김지환','이야']

print(len(a))

# 4
```

### for i in range(len(a))

- range(시작, 끝, 증감식) = 해당 조건 만큼 반복문 실행
- range(len(a)) ==> range(4) : 즉 0~3에 해당하는 반복을 실행

```py
for i in range(len(a)):
    print(i)

# 0
# 1
# 2
# 3
```

### 이중반복분

```py

arr = [[1,2,3],"4",[5,6]]

for i in arr:
    print(i)

# [1, 2, 3]
# 4
# [5, 6]

for i in arr:
    for j in i :
        print(j)

# 1
# 2
# 3
# 4
# 5
# 6

즉, arr의 값들을 i가 받고, i의 값 속에서 j 반복문으로 다시 한 번 값을 출력

```

#### arr에 들어있던 값 중 "4" 가 4(int)였다면 반복문이 실행 됐을까?

- `for i in arr: `는 실행 가능 // 단순히 arr의 값을 출력하기 때문
- 하지만 `for j in i: ` 는 실행 불가능 // int 값은 반복문 실행이 안되기 때문 // str, tuple, list, set, dict 만 가능 (set은 값을 자동정렬 후 출력 , dict은 key값을 출력)


#### index 값 구하기

```py

arr = [[1,2,3],"4",[5,6,7,8]]

for i in range(len(arr)):
    for j in range(len(arr[i])) :
        print(i,j ,"//", arr[i][j])

# 0 0 // 1
# 0 1 // 2
# 0 2 // 3
# 1 0 // 4
# 2 0 // 5
# 2 1 // 6
# 2 2 // 7
# 2 3 // 8

1. arr의 length = 3 ([1,2,3] / "4"/ [5,6,7,8])
2. range(len(arr)) == range(3) ==> 0 ~ 2 까지 3회 반복
3. i는 0 ~ 2 까지의 값을 받음
4. arr[i]는 해당 배열의 값 // arr[0] == [1,2,3] / arr[1] == "4" / arr[2] == [5,6,7,8]
5. len()은 내용 요소의 길이 반환 / len(arr[0]) == len(3) // len(arr[1]) == len(1) ...
6. range()에 len 값이 들어감 range(len(arr[0])) ==> 3번 반복
7. j가 값을 반환(숫자)
8. 반복문이 실행되며 print를 통해 i와 j 의 값이 출력 됨, arr[i][j]에 해당하는 값이 출력됨
```
