**함수**

문자열의 인덱스 계산 ("0 안 1 녕 2 하 3 세 4 요 5 ") -> length 값과 같음
배열의 인덱스 계산 ['안','녕','하','세,',요'] -> lenght - 1 값과 같음
                    0    1    2    3   4

- 함수명.length - 문자열 길이 도출 (문자열만 가능)
 
- 함수명.replace('기존값','변경값') - 문자 변경 (첫 번째만 변경)

- 함수명.replaceALL('기존값','변경값') - 문자 변경 (해당되는 모든 문자 변경)
 
- 함수명.substring(1, 4) - 인덱스의 1번째 글자 부터 4번째 글자 까지 출력
  *꿀팁* 어떠한 문서의 끝 글자 인덱스를 찾기 쉬움
  *주의* "0 G 1 o 2 o 3 d 4 ~ 5" 의 경우 로 계산해야함. 보이는 순서대로 생각하면 안됨
        문자열만 가능
  
- 함수명.slice(1, 5) - 인덱스의 1번째 글 부터 5번째 글 까지 출력한다.
  함수명.slice(1) - 인덱스의 1번째 글 부터 끝까지 출력한다.
  함수명.slice(-4) - 인덱스의 끝에서 4번째 글까지 출력한다.
  
- 함수명.split('글자') - 총 문자에 해당하는 글자를 제외하고 남은 것을 출력한다
 *주의* var a = "가나다가나다가나다" 일 때 a.split('나다') 를 입력하면
       배열 ['가','가','가','']로 출력. lenght는 4지만 인덱스값은 0,1,2,3 이다.(배열이니까)
       split 결과 배열의 마지막 요소가 빈 문자열("")이므로,
       slice로 출력할 때 length - 1을 사용하면 빈 값을 제외할 수 있다.
       a.split('') 를 입력하면 문자열 하나하나를 분리시킨다.
  
- 함수명.includes('*문자*') - 문자열 안에 해당 문자가 있는지 여부 출력
  
- 함수명.trimStart() - 왼쪽 공백 제거
  함수명.trimEnd() - 오른쪽 공백 제거
  함수명.trim() - 양쪽 공백 제거
