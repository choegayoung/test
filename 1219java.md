순서가 있음

[기본타입]
number    : 숫자(정수, 실수)
bigint    : 정수 -(2의 53승 -1) ~ 2의 53승-1
string    : 문자열 /*숫자와 정수를 포함할 수 있는 범위 (특수기호도 포함)*/

number와 bigint는 연산을 위해 따로 관리. string은 연산 불가.

true, false : 값 사이에 == 을 넣으면 자료형 안의 내용만 연산, ===을 넣으면 자료형가지 연산
ex) 5=="5" true, 5==="5" false

undefined : 초기화되지 않은 상태, 변수 초기화에 사용 
            함수에 입력값은 있지만 출력값이 없는 상태
            !!자바에서만 사용!!

NaN       : Not a Number 숫자가 정의되지 않은 상태 (자료형에 포함)

  변수
  var 변수명: /*한글로 선언가능*/
  typeof 는 문자열로 출력

  var num; - 변수 값이 지정되지 않았음
  ('텍스트' +num); - undefined
  ('텍스트', typeof num) - undefined

  var은 너무 광범위하게 입력이 가능해서 예측이 어려움
  let / const 사용.
  let - 변환할 수 있는 변수 입력
  const - 상수 입력

var 값을 수로 지정하여서 문자열과 통일이 안 될 경우
`텍스트${ a + b}` 과 같이 백틱으로 묶어주면 됨

 함수명.length - 문자열 길이 도출 (문자열만 가능)
 함수명.replace('기존값','변경값') - 문자 변경 (첫 번째만 변경)
 함수명.replaceALL('기존값','변경값') - 문자 변경 (해당되는 모든 문자 변경)
 함수명.substring(1, 4) - 인덱스의 1번째 글 부터 4번째 글 까지 출력 (인덱스는 글자만 인정. 특수기호 x)
