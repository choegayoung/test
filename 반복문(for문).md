배열을 사용할 땐 (값을 넣거나 빼낼때) 반복문을 사용.
**반복문 기본 문법**

for( let i = 0 /*변수 선언 및 초기화*/; i < 1 ; i = i + 1 ) 
for('반복할 내용';'언제까지 반복 할건지';'내용을 증가시킬지, 감소시킬지')

 - '언제까지 반복 할건지' -> true의 내용이 fulse가 나올 때 까지 반복한다. (논리를 입력해야한다)

//1. 변수 선언 및 값 설정
for (/*2.변수 선언 및 값 설정*/;;) {
1번 변수 사용 가능, 2번 변수 사용 가능
}
//1번 변수는 사용 가능(전역변수) //2번 변수는 사용 불가 (지역변수)
변수 선언은 var, let 만 가능, const는 불가 (변화해야 하기 때문)

**예제문제**
const a = [1,2,3,4,5];

for ( let 인덱스 = 0 ; 인덱스 < a.length ; 인덱스 = 인덱스 + 1 ) { console.log(`배열${인덱스} : ${a[인덱스]}`); }

내가 틀린 이유 - 1. 출력할 변수를 선언하지 않음.(const a 변수로만 계산하려고 했음 for( a = 0; a= a<length; a = a+1) 이렇게씀)
                2. 개념정리가 안됨 ( 인덱스 값을 구하려고 a[]를 사용하려고 함.
                3. /*for( let i = 0 /*변수 선언 및 초기화*/; i < 1 ; i = i + 1 ) */ 이 수식을 그냥 개무시하고 있었음. 뺨 한대 맞을 것.
                4. for 중간부분에 i < i.length 이러고 앉음. a의 길이를 재야하는데 등신임.
